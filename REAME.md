# AI-Assisted Dev Lab

A personal software lab for building practical, polished applications with AI-assisted development workflows.

This lab is not limited to one product category. It focuses on using modern coding agents and AI tools to move from idea → specification → implementation → debugging → documentation → portfolio-ready software.

## Purpose

The goal of this lab is to explore how far AI-assisted development can go when paired with clear product thinking, strong architecture, careful review, and high-quality UI/UX.

Projects in this lab are designed to be:

- useful enough to be real products
- polished enough for portfolio presentation
- structured enough to grow beyond prototypes
- documented clearly enough for others to understand

## What this lab builds

This lab may include:

- desktop applications
- web applications
- learning tools
- productivity tools
- dashboards
- developer utilities
- experimental product prototypes

The common point is not the app category, but the workflow: each project is designed and built with the help of AI/coding agents.

## Workflow

Typical project workflow:

1. Define the product vision
2. Write a detailed specification
3. Break the app into independent build phases
4. Use coding agents to implement each phase
5. Review, test, refactor, and polish manually
6. Document the final product with screenshots, architecture notes, and a roadmap

## Tools and Technologies

Common tools used in this lab:

- ChatGPT and Claude for planning, specification writing, review, and reasoning
- Coding agents such as Cursor, Windsurf, Claude Code, or Codex-style tools
- Git and GitHub
- TypeScript
- React
- Tauri
- Rust
- SQLite
- Tailwind CSS
- shadcn/ui
- Framer Motion
- GitHub Actions

Technology choices may change depending on the project, but the standard is always the same: clean architecture, high-quality UI, maintainable code, and honest documentation.

## Current Direction

The first major project direction is **Lexora**, a Windows-first, offline-first English–Vietnamese vocabulary learning platform with a Steam-inspired deck library, FSRS smart review, pronunciation tools, achievements, analytics, and a premium light-blue desktop interface.

## Lab Philosophy

AI can generate code quickly, but quality still comes from clear direction, strong constraints, careful review, and disciplined iteration.

This lab is built around that principle.
